# GoNode

#### A simple client server chat application using GO Node


### Features 

- Login i.e., username/password
- Public chat
- Private chat

### Start Client:
```sh
node chatclient.js localhost 8000
```


### Start Server:
```sh
go run ChatServer.go 8000
```
